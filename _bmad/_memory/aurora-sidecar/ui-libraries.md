## Common Libraries Aurora Uses

### Icon Libraries
- **Heroicons** - Tailwind's official icon set (https://heroicons.com)
- **Lucide** - Beautiful & consistent icon set (https://lucide.dev)
- **Simple Icons** - Brand icons (https://simpleicons.org)

### Component Libraries
- **21st.dev** - Premium UI components
- **shadcn/ui** - Re-usable components built with Radix UI and Tailwind
- **Headless UI** - Unstyled, accessible components

### CSS Frameworks
- **Tailwind CSS** - Utility-first CSS framework (default choice)
- **Custom CSS** - When design system requires unique styling

### Animation Libraries
- **Framer Motion** - React animation library
- **GSAP** - Professional-grade JavaScript animation
- **CSS Transitions** - For simple, performant animations
