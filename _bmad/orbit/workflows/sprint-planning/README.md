# Sprint Planning Workflow

**Purpose:** Plan sprint using Scrum methodology with backlog prioritization and capacity planning.

**Owner:** Project Coordinator (Nova)

**Inputs:**
- Product backlog
- Team capacity
- Sprint length (from config)
- Previous sprint velocity

**Outputs:**
- Sprint goal defined
- Sprint backlog created
- Tasks assigned to team members
- Sprint timeline established
- Email notifications sent

**Lifecycle Phase:** Deliver

**Status:** Workflow plan - implementation pending
